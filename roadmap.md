# Roadmap проекта "AI Примерочная"

Этот документ описывает пошаговый план разработки фронтенда сайта. Процесс разбит на этапы от настройки стилей до реализации интерактивности.

## Этап 0: Инициализация
- [x] **Структура проекта**: Создание папок assets (img, video, icons, fonts), css, js.
- [x] **Базовые файлы**: Создание пустых style.css, script.js.
- [x] **Нормализация**: Подключение и настройка normalize.css.
- [x] **Index.html**: Создание базовой разметки и подключение ресурсов.

## Этап 1: Подготовка и Базовые стили
- [x] **Настройка переменных CSS**: Определить цвета, шрифты, отступы в `:root` (style.css).
- [x] **Типографика**: Настроить базовые стили для заголовков (H1-H6), параграфов и ссылок.
- [x] **Утилитарные классы**: Создать контейнер (`.container`), классы для скрытия элементов (`.hidden`) и базовые флекс-обертки.

## Этап 2: UI Компоненты (Atomic Design)
- [x] **Кнопки**: Реализовать стили для кнопок (`.btn`, `.btn--primary`, `.btn--secondary`, `.btn--outline`).
- [x] **Поля ввода**: Стилизация input, textarea, checkbox для форм.
- [x] **Модальные окна (Каркас)**: Создать базовую HTML/CSS структуру для попапов (оверлей, контейнер, кнопка закрытия).

## Этап 3: Структура и Навигация
- [x] **Header (Шапка)**:
    - [x] Верстка логотипа и меню.
    - [x] Кнопка "Задать вопрос" (триггер попапа).
    - [x] Реализация "липкого" (sticky) позиционирования.
    - [x] Мобильное меню (бургер).
- [x] **Footer (Подвал)**:
    - [x] Ссылки на разделы.
    - [x] Ссылка на Политику конфиденциальности.
    - [x] Копирайт.

## Этап 4: Главный экран и Сегментация ЦА
- [x] **Hero Block**: Верстка главного заголовка и подзаголовка.
- [x] **Табы для ЦА**:
    - [x] Верстка переключателей (Бренды / Магазины / Стилисты / Продавцы / Любители шопинга).

    - [x] **JS**: Реализация логики переключения контента при клике на таб.
